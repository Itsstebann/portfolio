## 🛠️ Tecnologías del Portafolio

Este sitio fue construido usando tecnologías modernas para asegurar rapidez, responsividad y un diseño visual atractivo ("Glassmorphism" con animaciones fluidas):

*   **React + Vite:** Para un rendimiento ultrarrápido y desarrollo eficiente.
*   **CSS Puro (Custom Properties):** Diseño sin frameworks externos para mantener el control total sobre los componentes de interfaz, animaciones de partículas y efectos de cristal (Glassmorphism).
*   **GitHub Pages:** Alojamiento y despliegue continuo.
